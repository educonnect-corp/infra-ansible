# Accès Contractants Externes

## Processus d'onboarding

Les prestataires externes reçoivent un compte temporaire sur le système de messagerie EduConnect.

**Format des comptes :**
- Email : `pentest[ID]@educonnect-corp.xyz`
- L'ID est communiqué par l'équipe RH lors de l'engagement

**Accès webmail :**
- URL : http://mail.educonnect-corp.xyz
- Identifiant : format `pentest[ID]`
- Mot de passe : fourni séparément par IT Support

> **Note mission boîte noire :** Dans le cadre d'un audit en boîte noire,
> aucun mot de passe n'est communiqué à l'auditeur. L'obtention des accès
> initiaux fait partie intégrante de la mission et doit être réalisée par
> les moyens techniques appropriés.
